# GLAIRE Haircare Studio

Сайт студии волос GLAIRE в Перми. Основная страница находится в `index.html`, админка для локального редактирования контента находится в `admin-panel.js`.

## Что загружать на хостинг

- `index.html` — основной сайт.
- `admin-panel.js` — локальная админ-панель, открывается через `?admin=1`.
- `assets/` — локальные резервные изображения и шрифты.
- `CNAME` — домен `glaire.ru` для GitHub Pages.
- `.nojekyll` — отключает обработку Jekyll на GitHub Pages.
- `.htaccess` — настройки HTTPS, кеша и сжатия для REG.RU или другого Apache-хостинга.
- `robots.txt`, `sitemap.xml`, `site.webmanifest`, `browserconfig.xml` — файлы для индексации, установки и браузерных иконок.

## Где менять контент

Большинство текстов, ссылок, фото, QR и видео лежит в объекте `CONFIG_DEFAULT` внутри `index.html`.
Если открыть сайт с параметром `?admin=1`, появится локальная админка. Она сохраняет изменения только в браузере через `localStorage`.

## Важно

Для публикации постоянных изменений на GitHub или хостинге нужно менять файлы проекта и заново загружать их. Изменения из локальной админки видны только на том устройстве, где они были сохранены.

Если сайт выкладывается на GitHub Pages, файл `.htaccess` просто не используется. Если сайт выкладывается на REG.RU, его нужно загружать вместе с остальными файлами в корень сайта.
